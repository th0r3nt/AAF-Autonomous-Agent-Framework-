## MENTAL STATE
...показывает последние 5 сущностей из SQL таблицы MentalState...
Формат:
имя_сущности: ... (допустим, th0rent)
важность_сущности: ... (допустим, high)
текущий_статус: ... (допустим, спит)
описание сущности: ... (допустим, 'создатель AAF')
контекст: ... (допустим, 'просил не будить')

пометка: нужно как-то настроить, чтобы добавлялся блок 'связанная_информация':
Например, если у меня есть аккаунт в телеграме и агент об этом знает - тут показывается 'профиль в Telegram: @username'
Это позволит связывать сущности и получить консистентность памяти
